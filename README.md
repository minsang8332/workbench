## 실행하기

> node 16.20.1^ 18.20.2^

💡**.env** 환경 변수 파일이 있어야 실행 및 설치 가능함

```
// 패키지 설치
npm run setup

// 개발하기
npm run serve

// 윈도우 어플리케이션 빌드
npm run build

// 윈도우 어플리케이션 배포 (환경변수 GH_TOKEN 필요)
npm run deploy

// electron 전용 아이콘들로 일괄 생성
npm run icon
```
