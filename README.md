## 설치하기

> node 16.20.1^ 기준

```
// 설치하기
npm run setup

// 개발하기
npm run serve

// 빌드하기
npm run build
```

### 그 외 부가 명령어

```
// input 경로의 아이콘을 electron 전용 아이콘들로 일괄 생성 해줌
npm run icon

// 출시하는 단말기 환경변수 GH_TOKEN 이 선언되어 있어야함
npm run deploy
```
